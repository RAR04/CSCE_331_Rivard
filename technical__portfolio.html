A+ Computer Science

I worked internships at A+ Computer Science developing tests on the Java programming language designed to prepare students for the Texas UIL State programming competition. The tests range in difficultly, from basics of programming and Java syntax to data structures and algorithms, graph theory, and niche Java semantics. I worked with other interns to edit and review the tests that we all created. The first internship, I also helped test a new software solution that allows users to program directly in the company's website.

Bio ISAC
